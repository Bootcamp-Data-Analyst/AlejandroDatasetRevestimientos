# Proyecto de análisis básico de ventas de revestimientos

Este proyecto es mi primer acercamiento al análisis de datos con Excel.  
A partir del archivo `revestimientosQuery.xlsx`, he aprendido a organizar información, crear tablas dinámicas y construir un dashboard sencillo para entender mejor los datos.

El archivo contiene datos de ventas de distintos productos de construcción (como perlita, mortero y yeso). Cada fila representa una venta, e incluye cosas como:

- la fecha,
- el cliente,
- el producto vendido,
- el precio,
- el coste,
- el beneficio,
- la zona,
- y el comercial que hizo la venta.
- ...
  


---

## ¿Qué he hecho en este proyecto?

### 1. Trabajar con una tabla de datos  
Primero he revisado el dataset, que trae información cruda. He aprendido a:

- moverme entre las columnas,
- entender qué significa cada dato,
- y ver cómo se relacionan entre sí.

Esto me ha servido para comprender mejor el tipo de información que se usa en las empresas para ver cómo están yendo las ventas.

---

### 2. Crear una tabla dinámica  
Con la tabla dinámica he podido:

- resumir los datos de forma automática,
- agrupar ventas por producto, zona o cliente,
- y calcular totales sin tener que hacer fórmulas difíciles.

Aquí es donde he empezado a ver qué productos se venden más o qué zonas compran más.

---

### 3. Construir un dashboard  
También he creado un pequeño dashboard en Excel que permite:

- filtrar los datos por año, zona o producto,
- ver gráficos que cambian según lo que seleccione,
- y mostrar de manera visual las ventas y beneficios.

Aunque es mi primer dashboard, me ha ayudado muchísimo a entender cómo presentar datos de forma clara y útil.

---

##  ¿Qué he aprendido?

Con este proyecto he aprendido cosas básicas pero muy importantes:

- cómo se limpia y revisa un dataset,
- cómo se usan tablas dinámicas,
- cómo se conectan segmentaciones (filtros) a gráficos,
- y cómo hacer un dashboard sencillo pero funcional.

Es un primer paso en el mundo del análisis de datos y me ha servido para perderle el miedo a Excel y a los datos grandes.
